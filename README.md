# BLOG-App


### Features
1. Log in and Sign up for user.
2. View published Blogs.
4. Write a new Blog after Log in.
6. Form Handling


___
### To run Frontend Application,
```
git clone https://github.com/Manoramsharma/Blog-Frontend-React.git
```

```
cd client
```

```
npm install
```

```
npm run start

```
